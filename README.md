#Ethara.tasktrack
🗂️ What I Built
 A Team Task Tracker that enables:
Admins to create projects, manage members, assign tasks, and track progress
Team members to collaborate, manage workflows, and update task statuses efficiently

⚙️ Tech Stack
Frontend - React + Vite
Backend - Node.js + Express
Database - MongoDB + Mongoose
Authentication - JWT + bcrypt
Authorization - Role-Based Access Control (RBAC)
Deployment - Vercel

✨ Core Features
  Secure JWT Authentication
  Role-Based Access Control
  Project & Task Management
  Kanban Board Workflow
  Live Dashboard & Progress Tracking
  RESTful API Architecture
  Responsive Modern UI
  Fully Deployed Live Application
One of the best parts of this assessment was learning how to think and build under pressure - making fast architectural decisions, debugging efficiently, and focusing on delivering a polished real-world product instead of just writing code.

This project helped strengthen my understanding of:
Full-stack architecture
Secure authentication systems
API design
Database modeling
Deployment workflows
Scalable frontend structure
